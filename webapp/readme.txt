说明：这一个 node + mongodb + gm 搭建的小的 blog 系统
	后面可能要把 gm 删除掉


启动：
	1. 启动 gm 
		在 cmd 的 DOS 窗口地位到 gm 的安装目录，找到 gm.exe 所在的目录，执行 >gm.exe

	2. 启动 mongodb
		在 cmd 的 DOS 窗口下，启动 mongodb ，注意启动是当前 cmd 所在的目录 >mongod --dbpath F:\NodeBlog\MoDB

	3. 启动 node
		在 cmd 的 DOS 窗口下，定位到当前项目中 app.js 所在的目录，然后执行启动 node >node app.js
